# Frontend

React frontend built with Vite.

## Scripts

### `npm start`

Runs the Vite development server on `0.0.0.0`.

### `npm run dev`

Alias for the Vite development server.

### `npm run build`

Builds the production bundle into `dist`.

### `npm run preview`

Serves the production build locally.

### `npm test`

No test runner is currently configured.

## Notes

The `@` import alias resolves to `src`.

Set `ENABLE_HEALTH_CHECK=true` when running the dev server to expose:

- `/health`
- `/health/simple`
- `/health/ready`
- `/health/live`
- `/health/errors`
- `/health/stats`
